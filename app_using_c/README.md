# 🎮 Retro To-Do List App

A stylish terminal-based to-do list app with a retro gaming aesthetic, featuring typewriter text effects and vibrant colors!

## ✨ Features

- **Typewriter Effect**: Text reveals character-by-character like classic video games
- **Colorful UI**: Beautiful cyan, green, and colored text with box borders
- **Check/Uncheck Tasks**: Mark tasks as complete with checkboxes
- **Simple Commands**: Add, view, check, and remove tasks easily
- **Retro Aesthetic**: Designed for that nostalgic gaming feel

## 🚀 Quick Start

```bash
# Compile
gcc -Wall -Wextra -std=c99 -o todo_app main.c

# Run
./todo_app
```

Or use make:
```bash
make
./todo_app
```

## 🎯 Usage

1. **Add Task** - Create new to-do items (e.g., "brush my teeth in the morning")
2. **View All Tasks** - See all your tasks with cool animations and checkboxes
3. **Check/Uncheck Task** - Mark tasks as complete ☑ or incomplete ☐
4. **Remove Task** - Delete tasks by number
5. **Exit** - Close the app
